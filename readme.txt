This plugin helps users to easily add the contact form to their wordpress site by widget, page, post.

== Description ==
<ul>
	<li>Add contact form to widget, page, post</li>
	<li>Form includes: name, email, subject and message</li>
	<li>AJAX submit message</li>
	<li>(New) Send email to configured email list (Wordpress must have permission to send mail)</li>
	<li>(New) Multi-languages support</li>
	<li>Support notification when submit message:
	<ul>
		<li>Check required fields</li>
		<li>Check email format</li>
		<li>Check over-date status if user leave form too long (> 10 minutes)</li>
	</ul>
	</li>
	<li>Admin panel: 
	<ul>
		<li>Page navigator</li>
		<li>Messages include READ and Un-READ messages</li>
		<li>Support search only 'un-read' message</li>
		<li>Message will be READ when user open it</li>
		<li>Support deleting each message </li>
		<li>Support adding note to each message</li>
	</ul>
	</li>
</ul>

Visit us: <a href="http://webfacescript.com/category/product-list/product-list-wordpress-plugins/" target="_blank">webfacescript.com</a>

== Installation ==

1. Upload the entire wfs_simple_contact folder to the /wp-content/plugins/ directory or Add New Plugin by upload the wfs_simple_contact.zip
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'WFS Simple Contact' menu in your WordPress admin panel.
